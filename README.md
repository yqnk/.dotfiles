# .dotfiles
> My ArchLinux dotfiles.

## Install
```sh
$ git clone git@github.com:yqnk/.dotfiles.git ~/dotfiles
$ cd

# Install wallpapers
$ ln -s ~/dotfiles/wallpapers ~/.local/share/wallpapers

# Install icons
$ ln -s ~/dotfiles/icons ~/.local/share/icons

# Install fonts
$$ sudo pacman -S --needed - < fonts.txt
```
